px

em

By default the font size of every browser is 16px. em is refered to the closest parent font size. In html document many tags are nested into each other. If many elements size is refered using em, it becomes very difficult to manage em unit.

When you define some child container width as %, it is measured relative to the amount of parent container element width

When should we use what???

  • font size=> rem
  • width=> % , often max width, optional vw, ch is a good idea too
  • height( think why first )=> min-height, %, rem, vh( be careful)
  • padding/ margin=> em/rem
  • media query=>em
