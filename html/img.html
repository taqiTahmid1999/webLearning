Image and Icon Learning

Things to remember

  • figure is the holding container that contains the content
  • figcaption give caption to the content
  • img is the self closing tag that can show the image in html
  • code is used to write any code in html document
  • URL is in the src
  • alt text will be shown if image could not load
  • When you put mouse cursor over img title text will be shown
  • height and width of the content
  • loading lazy will load the image when we reach to the image of the html document. It will ensure loading performance
